### 🎉简介

基于vite5.x + vue3.x + arco-design2.的前端框架youke-admin，欢迎使用！

### 特性

- **最新技术栈**：Vue 全家桶，基于`Vue3`, `Vite`, `Vue Router` 开发
- **UI**：全新前端框架`arco-design`
- **Mock 数据**：内置 Mock 数据方案 搭配`mockjs`,`vite-plugin-mock`插件食用
- **主题**：可配置的主题
- **布局**：内置多种布局方式，满足多方需求

### 快速开始

```bash

# 进入项目目录
cd youke-admin
```

```bash
# 安装依赖
npm i
```

```bash
#启动
npm run dev
```
```bash
#打包
npm run build:prod
```
